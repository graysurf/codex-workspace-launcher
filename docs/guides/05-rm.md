# `rm`

Removes workspaces in the selected runtime (`container` default, `host` optional).

## Remove one workspace

```sh
agent-workspace-launcher rm <workspace> --yes
```

## Remove all workspaces

```sh
agent-workspace-launcher rm --all --yes
```

## Host runtime fallback

```sh
agent-workspace-launcher --runtime host rm <workspace> --yes
```

## Container-only volume behavior

Use `--keep-volumes` only with `--runtime container`:

```sh
agent-workspace-launcher --runtime container rm <workspace> --yes --keep-volumes
```
